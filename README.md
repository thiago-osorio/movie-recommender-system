## 🆘 Description

Content Based Machine Learning Recommending System build with the power of [`cosine similarities`](https://en.wikipedia.org/wiki/Cosine_similarity).

## 💻 Technologies Used
### Web Technologies
- Python
- SQLite
- Heroku

### Machine Learning
- Numpy
- Pandas
- SciPy
